# Concept Art & Reference Images

This folder contains AI-generated reference images for Phase 8 visual development.

## Purpose
These images serve as style guides for:
- Color palettes
- Character design proportions
- Environment art style
- Overall visual direction

## Images We Need

### 1. Character Style Reference
**Prompt suggestion:**
```
Stardew Valley style pixel art characters, 16-bit, top-down view, Greek mythology theme,
simple colorful sprites, chibi proportions, white background
```

### 2. Tile/Environment Reference
**Prompt suggestion:**
```
Stardew Valley style 16x16 pixel art tiles, grass dirt stone water textures,
retro RPG game assets, top-down perspective, tileset sheet
```

### 3. NPC Sprites Reference
**Prompt suggestion:**
```
Greek mythology characters pixel art, Hermes Aeetes Daedalus Circe,
Stardew Valley RPG style, simple colorful designs, idle standing pose
```

## Usage
1. Place generated images in this folder
2. Reference when creating procedural assets
3. Use MiniMax MCP `understand_image` to analyze and extract color palettes/styles
